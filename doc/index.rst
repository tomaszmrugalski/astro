#####################
Perylune User's Guide
#####################

Perylune - a periapsis (a lowest point in orbit) around Luna (better
known as the Moon). Also a library of tools intended to aid various
calculations related to Keplerian orbits and orbital mechanics in
general. The software is in very early stages of development.

This is the reference guide for Perylune version |release|.

Links to the most up-to-date version of this document, along with other documents for
Stork, can be found on the `Perylune project homepage  <https://github.com/tomaszmrugalski/perylune>`_ .
